职位描述

岗位职责： 
1. 负责公司相关业务、产品和服务的后端开发和维护。 
2. 负责产品后端架构的设计、实现、改进和优化，确保后端服务稳定、可靠、高效。 
3. 参与后端前沿技术的探索。 
任职要求： 
1. 计算机基础知识扎实，对数据结构及算法有较好的掌握。 
2. 两年或以上 Python/Java/Go 语言后端开发经验，有 Python 经验者优先。 
3. 熟悉 MySQL 数据库以及 Redis 或 Memcached、MongoDB 等 NoSQL 技术。 
4. 有一定的系统设计能力及良好的抽象能力，能够对业务逻辑进行合理的抽象和拆分。 
5. 有优秀的解决问题的能力，有很强的学习能力。 
6. 具有大数据分析处理、AI 产品落地经验者优先
